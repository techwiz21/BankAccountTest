1.Run the project

2.Import the Postman Collection from https://github.com/techwiz21/BankAccountTest

3.In the Postman Collection there are 4 different requests defined :	4.1:Create Bank Account(POST)
									4.2:Deposit Money(PUT)
									4.3:Get Account Details(Get)
									4.4:Delete Account(DELETE)

4.1:It accepts a json string with a specified currency and an amount of money to create the account with.

4.2:It accepts a json string with a specified currency and an amount of money to deposit in that specified currency account

4.3:It accepts a currency parameter to return the amount of money the user has in the account

4.4:It accepts a currency parameter to delete the specified account
